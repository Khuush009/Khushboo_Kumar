Phoenix ILS integrated platform

This was a project I started back in January 2017. I was in the team of 15 persons.

The project is an ordering and billing management system for a quality assurance service provider. The application is developed using JAVA language and Spring and Hibernate technologies

Testing is carried out for various industries. The tests to be performed are recorded and the pricing logic is applied on it considering discounts, VAT, TAX to generate the invoice in PDF format using IREPORT tool. The application is widespread across various countries and is customized according to the region.

Technologies:
  • - JAVA
  • - Spring MVC + Hibernate Framework
  • - Eclipse
  • - Oracle SQL Developer
  • - IReport - PDF generator
